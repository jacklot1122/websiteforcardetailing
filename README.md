# T&W Auto Detailing Website

Professional auto detailing website for T&W Auto Detailing, serving Central Coast & Hunter Area.

## Features

- Modern, responsive design
- Dark theme with gold accents
- Mobile-friendly navigation
- Service pages with detailed pricing
- Contact form
- Ceramic coating packages
- Paint correction services
- Interior & exterior detailing

## Pages

- **Home** - Hero section, services preview, gallery
- **About Us** - Company story and values
- **Services** - Detailed service descriptions
- **Prices** - Complete pricing for all services
- **Contact** - Contact form and information

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Responsive Design
